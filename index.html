🚀 Full Setup Guide Part A — Enable Google Cloud Vision API 1. Go to console.cloud.google.com
2. Select your project (same one linked to your Apps Script)
3. Search "Cloud Vision API" → click it → click Enable
4. That's it — Apps Script uses your account's OAuth automatically, no key needed

Part B — Deploy Apps Script 1. Go to script.google.com → open your project
2. Replace ALL code with the new Code.gs file
3. Line 14: set your Spreadsheet ID
4. Function dropdown → testDriveAccess → Run ▶ → authorize everything
5. Deploy → New deployment · Web app · Execute as Me · Access Anyone
6. Copy the URL and paste it above
