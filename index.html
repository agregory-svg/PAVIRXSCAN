📖 Setup guide — tap to expand
Part 1 — Google Service Account 1. console.cloud.google.com → your project
2. Enable: Cloud Vision API, Google Drive API, Google Sheets API
3. IAM & Admin → Service Accounts → Create → name it rxscan → role: Editor
4. Click the account → Keys → Add Key → JSON → download
5. Open JSON in text editor → copy all → go to base64encode.org → encode → copy result
6. Share your Google Sheet with the service account email → Editor

Part 2 — Render Deployment 1. Create a private GitHub repo called rxscan-backend
2. Upload main.py and requirements.txt
3. render.com → New → Web Service → connect repo
4. Runtime: Python 3 · Build: pip install -r requirements.txt
5. Start: gunicorn main:app --bind 0.0.0.0:$PORT --workers 1 --timeout 120
6. Environment variables: SPREADSHEET_ID, GOOGLE_CREDS_B64, SHEET_NAME=Sheet1
7. Deploy → copy URL → paste above

⚠ Free Render sleeps after 15 min. First use takes ~30-60s to wake. The app handles this automatically.
