🚀 One-Time Backend Setup (~15 min total) Part 1 — Google Service Account (5 min) 1. Go to console.cloud.google.com → your project
2. Enable these 3 APIs: Cloud Vision API, Google Drive API, Google Sheets API
3. Go to IAM & Admin → Service Accounts → Create Service Account
4. Name it rxscan, click Create, skip optional steps
5. Click the new service account → Keys → Add Key → JSON → download the file
6. Open the JSON file, copy ALL its contents
7. Go to base64encode.org, paste the JSON, encode it → copy the result (this is your GOOGLE_CREDS_B64)
8. Share your Google Sheet and Drive folder with the service account email (looks like [email protected]) — give it Editor access

Part 2 — Deploy on Render (5 min) 1. Go to github.com → create a new private repo called rxscan-backend
2. Upload main.py and requirements.txt to this repo
3. Go to render.com → sign up free → New → Web Service
4. Connect your GitHub account → select rxscan-backend
5. Settings:
  • Runtime: Python 3
  • Build command: pip install -r requirements.txt
  • Start command: gunicorn main:app --bind 0.0.0.0:$PORT --workers 1 --timeout 120
6. Under Environment Variables add:
  • SPREADSHEET_ID = your Google Sheet ID
  • GOOGLE_CREDS_B64 = the base64 string from step 7 above
  • SHEET_NAME = Sheet1 (or your tab name)
7. Click Deploy — takes ~2 min. Copy the URL (e.g. https://rxscan-xxxx.onrender.com)
8. Paste that URL above

⚠ Free Render tier sleeps after 15 min of inactivity. First request after sleep takes ~30 seconds to wake up. This is normal.
